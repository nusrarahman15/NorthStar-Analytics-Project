# NorthStar-Analytics-Project

Integrated Analytics Solution: SQL in R | R Analytics | Python | MongoDB

Project Overview

NorthStar Urban Mobility and Logistics is a multi-service transport and delivery service provider, covering shuttle transport, last mile delivery, EV charging, warehouse dispatch as well as mobile platform services. This project is aimed to create a comprehensive analytics solution that detects inefficiencies in operations, enhances the quality of service and enables data-driven decision-making.


 Business Challenges
 
 The organisation has a number of operational challenges:

1.	A rise in the number of deliveries which fail or are delayed.
2.	Rising customer complaints
3.	Costs are higher than income, and increasing in proportion.
4.	Data silos in various systems
5.	Difficulty between structured and semi-structured data.
6.	Poor interoperability between structured and semi-structured data.


 Project Objectives
 
The project's objectives are to:

1.	pinpoint major inefficiencies within operational zones and hubs
2.	Identify delivery failures and patterns of incidents
3.	Assess customer complaints, and financial effects
4.	Explore the interdependence of drivers, routes and incidents
5.	Create an extensible NoSQL solution for event driven data.
6.	Allow for integrated reporting and decision making


 Technologies Used

SQL in R

•	Querying data and relational analysis
•	Failure rate and performance analysis.
•	Operational reporting

R Analytics

•	Statistical analysis with the package dplyr
•	Creating data visualisations using ggplot2
•	The normal method is to use correlation and regression modelling.

Python

•	Data cleaning and data transformation
•	Feature engineering
•	Advanced analytics and visualisation.

MongoDB Atlas

•	NoSQL database design
•	Event-driven and semi-structured data storage.
•	Aggregation pipelines are supported, as well as CRUD operations.

Google Colab

•	Development and execution environment.

 GitHub

•	Version control
•	Organising and working on the project together



 Conclusion

This project is an example of how SQL, R, Python, and MongoDB can be used together to solve real-world business problems. NorthStar can help optimise business operations, boost customer satisfaction and facilitate sustainable business growth by integrating fragmented data sources and utilising advanced analytics.
